# BeamSolver

A professional-grade tool for calculating beam deflections, shear forces, and bending moments.

## Features
- Shear Force Diagrams
- Bending Moment Diagrams
- Deflection Curves
- Support for multiple loads and supports

## Installation
```bash
npm install
```

## Development
```bash
npm run dev
```

## Testing
```bash
npm test
```

## Build
```bash
npm run build
```
